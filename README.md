# Scintillators for Low-Cost Pet
Documentation for a project to develop low-cost nanocomposite scintillator pixels for a PET scanner prototype. The purpose for these notes is primarily as a means of collating findings, data and progress into one piece of IP to facilitate continuity between different stages of the project. This repository consists of the following categories.
## Context
Details on the relevant theory, methodology and progress are stored in the root directory. Key notes are listed below, with links to relevant subsidiary notes embedded in them.  
- [Introduction](Introduction.md)
- [Positron Emission Tomography](PositronEmissionTomography.md)
- [Scintillator Landscape](Scintillator%20Landscape.md)
- [Scintillator Design Considerations](Scintillator%20Design%20Considerations)
- [Characterization Methods](Characterization)
## References
A growing list of [References](References.md) to useful resources is included as a kind of database of the literature consulted during the course of the project.
## Logbook
A list of the pixel fabrication techniques and trials that have already been attempted can be found in the [Fabrication Log](Logbook.md), which should be updated with any new developments.
## Manuals
A folder containing documentation and manuals (listed in [Manuals](Manuals)) for essentially any externally procured products. This includes any measurement instruments, industry standard scintillator pixels, chemical safety datasheets, quotes etc. 
## Figures
A folder to store any generated figures, both for ease of access and for embedding in these notes. 
## Data
A folder for all data taken during characterization and experimentation. 
## Code
The codebase used for this project, including data analysis, instrument control and figure generation. [Documentation](Documentation) for the code is currently in progress.
## Progress Reports
Biannual reports detailing the last six months of progress. Typically to be submitted to funders.
